用途MCP 让通用 Agent 直接调用 PagePilot：发布、更新、管理屏幕、下发刷新/休眠/关机、请求截图。
权限发布可匿名或 Token；屏幕相关操作必须使用注册用户 Token，设备 Token 不会暴露给 Agent。
服务器安装后保存 PagePilot 服务器地址，Token 可永久或按有效期自动失效。
hostctl-mcp --server https://pagepilot.example.com --token YOUR_TOKEN
发布/更新deploy_site 支持单 HTML、多文件、公开/未公开、访问密码；更新必须传已有 code 并追加版本
匿名绑定claim_anonymous_session 把 anon:{sessionId} 发布的站点迁移到当前注册用户
站点管理list_sites、set_access_password、版本查询、可见性和删除等能力沿用同一套权限
